# Travel-Booking-Console-System-C-OOP-
A console-based application developed using C++ to simulate real-world travel agency operations. 


Designed an interactive system to manage customer records, cab bookings, hotel reservations, and billing services.            
– Implemented core OOP concepts: Encapsulation, Inheritance, Abstraction, and Polymorphism.                                    
– Integrated file handling for persistent storage of customer and billing data, ensuring modularity and scalability.
– Developed a reusable and extensible architecture for future additions like flight booking and online payments. 
– Technologies Used: C++, Object-Oriented Programming (OOP), File Hand
